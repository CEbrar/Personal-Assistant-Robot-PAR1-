# Components List

This file documents the main components planned for the PAR1 Personal Assistant Robot project.

## Main Controller

- ESP32-S3 DevKitC-1 development board

## Input Components

- 2 x INMP441 I2S digital microphone modules

## Output Components

- MAX98357 I2S mono speaker amplifier
- 8 Ohm 1W-2W mini speaker
- 0.96 inch I2C OLED screen

## Movement Components

- 2 x SG90 mini servo motors
- Mini pan-tilt bracket kit

## Power Components

- 1200mAh LiPo battery
- TP4056 lithium battery charging module
- MT3608 DC-DC step-up voltage booster

## Wiring and Connection

- Male-female jumper cable set

## Connectivity

- Wi-Fi connection through the ESP32-S3 board
- AI API integration, such as Gemini or another AI service

## Mechanical Design

- 3D-printed robot body
- Mounting parts for the ESP32-S3 board, servo motors, microphones, speaker, amplifier, OLED screen, and battery module

## Notes

The component list may change during the development process. Any major change should be documented in the development log.

Power-related components such as the LiPo battery, TP4056 charging module, and MT3608 voltage booster should be tested carefully before being connected to the full robot system.
