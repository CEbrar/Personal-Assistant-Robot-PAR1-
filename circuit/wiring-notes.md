# Wiring Notes

This file will document how each electronic component is connected to the ESP32-S3 DevKitC-1.

## Components to be connected

- ESP32-S3 DevKitC-1
- INMP441 I2S digital microphone modules
- MAX98357 I2S mono speaker amplifier
- 8 Ohm 1W-2W mini speaker
- SG90 mini servo motors
- 0.96 inch I2C OLED screen
- TP4056 lithium battery charging module
- MT3608 DC-DC step-up module

## Notes

The exact GPIO pins will be added after the first circuit tests.
